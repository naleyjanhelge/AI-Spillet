Did NOX go quiet?

Open a public support request with a short, non-sensitive description of the problem, your iPhone model, iOS version, and the Prompt Heist version shown in the App Store.

Open support request

Before contacting support

  • Confirm that you are using an Apple Intelligence-compatible iPhone with iOS 26 or later.
  • Open iOS Settings and make sure Apple Intelligence is enabled.
  • If the model is still downloading, connect to Wi-Fi and power, then try again later.
  • Restart Prompt Heist after changing Apple Intelligence settings.
  • If only scores are missing, check that you are signed in to Game Center.
GitHub support requests are public.

Never include names, email addresses, passwords, financial information, private workplace data, NOX conversations, or other personal, sensitive, or confidential content. Check screenshots for notifications or identifying details before posting.

Reset progress

Settings → Reset all progress removes unlocked levels and best scores stored by Prompt Heist on this iPhone. This cannot be undone. Game Center information is managed separately through Apple.

Privacy questions

Read the Privacy Policy for details about on-device NOX conversations, local progress, and optional Game Center features. Prompt Heist has no developer account or developer-operated game server.
