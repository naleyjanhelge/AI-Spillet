# Prompt Heist 2.1.0 (Build 7)

## TestFlight notes

Prompt Heist is now easier to enjoy in short sessions:

- Added NOX Drills with 12 bite-sized challenges.
- Every drill supports multiple valid ways to outsmart NOX.
- Added clear, revocable consent before any message is sent to OpenRouter.
- Added AI privacy details and safer provider routing that denies data
  collection and response caching.
- Added Chill and unlockable Hard difficulty per drill.
- Added persistent route mastery, best scores, stars, and mastery XP.
- NOX now remembers trust, respect, friction, routes, and campaign decisions.
- Added route archives, improved Heist Board stats, and richer room feedback.
- Improved final ending flow, scene direction, reduced-motion behavior, and save compatibility.

## What to test

1. Open NOX Drills from the facility screen and clear any challenge in Chill mode.
2. Confirm Hard mode unlocks for that challenge and requires additional proof.
3. Replay a drill through another route and confirm route mastery and XP update.
4. Resume a campaign room after closing and reopening the app.
5. Complete or replay rooms through different routes and check NOX continuity on the Heist Board.
6. Verify Daily Breach, Game Center, reduced motion, and the final Open Core verdict flow.
